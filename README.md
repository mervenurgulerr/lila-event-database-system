# lila-event-database-system

## Project Overview
Lila Event is a relational database system designed to manage the complete lifecycle of cultural events, including event organization, ticket sales, participant tracking, and reporting.

## System Scope
The database system supports:

- Ticket sales and pricing
- Participant and artist management
- Sales agent tracking
- Rule-based event constraints
- Search and reporting functionality
